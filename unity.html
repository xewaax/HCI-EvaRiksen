Unity

Introductie

Unity is een programma waarin je virtuele omgevingen kan maken. Het is voornamelijk bedoeld om spellen in te maken. Ik heb een Unity handleiding gevolgd en zo mijn eigen omgeving gemaakt met bepaalde objecten en triggers. Hieronder zijn mijn stappen te vinden.

Stappen

Ik ben begonnen met het maken van een terrein. Met verschillende tools kan je heuvels en bergen implementeren. Dit heb ik gedaan en ik heb ook een gratis sneeuw texture gedownload in de asset store. Dit was niet heel mooi maar ik vond het beter dan wanneer alles helemaal wit was.

De volgende stap was het toevoegen van een player. Dit is eigenlijk een first person camera. Nu kon ik echt rondlopen in de game.

Hierna heb ik een aantal objecten toegevoegd. Je kan objecten in Unity bepaalde componenten meegeven. Zo kan iets bijvoorbeeld een vaststaand object zijn, of een object dat je omver kan duwen. Ik heb een aantal objecten neergezet. De kubus heeft hierbij als eigenschap dat ik hem om kan duwen als ik er tegenaan duw.

Je kan ook objecten downloaden uit de asset store in Unity. Ik heb een auto gedownload van de asset store.

Aan deze objecten kan je ook triggers toevoegen. Dit doe je met behulp van scripts. Ik heb een script toegevoegd zodat er een vlammetje boven de auto ontstaat als ik dichtbij kom. Mijn auto is helaas een beetje groot, dus je ziet het pas echt goed als je met de player bovenop de auto gaat staan!
